# Useful navigation keys (Only Active in Vi Mode)
- Go to the top of the page (Without Scrolling)
    - `Shift + H`

- Go to the middle of the page (Without Scrolling)
    - `Shift + M`

- Go to the bottom of the page (Without Scrolling)
    - `Shift + L`
